# Operators

| Sign | Name              | Operation          | ASCII Code     |
| ---- | ----------------- | ------------------ | -------------- |
| +    | Plus              | Add                | Alt+43         |
| -    | Minus             | Subtract           | Alt+45         |
| \*   | Star              | Multiply           | Alt+42         |
| /    | Slash             | Division           | Alt+47         |
| ^    | Circumflex Accent | Potency            | Alt+94         |
| ()   | Parenthesis       | Operation Priority | Alt+40, Alt+41 |

# Priority

- Parenthesis
  - Potency
    - Multiply
      - Addition
      - Subtraction
    - Division
      - Addition
      - Subtraction
